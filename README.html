代码编写规范 ​

  • vue文件代码遵循如下结构：
vue
<script setup lang="ts"></script>

<template></template>

<style scoped>
@reference '../style.css';
</style>
  • 代码尽量使用 tailwindcss 类名，以 @apply 指令组合原子类实现特定效果，尽量减少手动纯写 css 代码，例如：
css
.hero__glow {
	@apply absolute rounded-full blur-[70px] opacity-[0.45] pointer-events-none;
}
